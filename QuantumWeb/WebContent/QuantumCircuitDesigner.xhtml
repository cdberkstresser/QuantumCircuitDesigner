This website is designed to provide for maximum user privacy and transparency. No sign-on is required, and this site does not use cookies nor does it track user information beyond the necessary cookies, access logs,and session IDs needed to persist circuits enough to edit them provided by the web service engine and the JSF/Bootsfaces framework.

This website offers a file open / save service. In order to maximize this transparency while providing users the ability to save and reload their work, the file format itself is xml and can be viewed in a plain text editor for those wishing to see the information being passed into and out of quantum circuits.

For further analysis of privacy practices, the source code of this web site can be found at https://github.com/cdberkstresser/QuantumCircuitDesigner.
